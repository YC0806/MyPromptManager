# 快速验证清单

插件已修复所有布局问题，使用此清单快速验证。

## 🔄 重新加载插件

```
1. 打开 chrome://extensions/
2. 找到 "MyPromptManager - AI History Sync"
3. 点击刷新按钮 🔄
```

## ✅ 验证清单

### 1. 对话同步标签页

**打开插件，默认显示"对话同步"标签**

- [ ] 页面顶部显示两个标签："对话同步"（高亮）和"Prompt库"
- [ ] 看到"同步状态"标题
- [ ] 看到三个状态项：当前提供商、自动同步、已保存对话
- [ ] 看到三个按钮：提取当前对话、同步所有对话、设置
- [ ] 底部看到"支持的 AI 提供商"信息
- [ ] **没有空白区域遮挡任何内容**

**测试设置功能**

- [ ] 点击"设置"按钮
- [ ] 设置区域展开，显示 API 地址输入框和自动同步开关
- [ ] 点击"取消"按钮，设置区域收起

### 2. Prompt库标签页

**点击"Prompt库"标签**

- [ ] 标签高亮切换到"Prompt库"
- [ ] 看到搜索框："搜索 Prompts 和 Templates..."
- [ ] 看到三个筛选按钮：全部、Prompts、Templates
- [ ] 看到列表区域（显示"加载中..."或 Prompt/Template 列表）
- [ ] **不显示设置区域**
- [ ] **不显示"支持的 AI 提供商"信息**
- [ ] **没有空白区域遮挡任何内容**

**如果后端运行且有数据**

- [ ] 看到 Prompt/Template 卡片列表
- [ ] 每个卡片显示标题、类型标签、描述、时间
- [ ] 卡片类型标签颜色正确（Prompt 蓝色，Template 紫色）

### 3. 标签切换

- [ ] 点击"对话同步"，内容切换到同步界面
- [ ] 点击"Prompt库"，内容切换到库界面
- [ ] 切换流畅，没有闪烁或错位

### 4. 模态框功能

**前提：后端运行且有数据，在"Prompt库"标签**

- [ ] 点击任意 Prompt/Template 卡片
- [ ] 模态框从屏幕中央弹出
- [ ] 模态框背景半透明遮罩覆盖整个界面
- [ ] 模态框显示标题、类型标签、时间、完整内容
- [ ] 模态框显示两个按钮：填入对话框、复制
- [ ] 点击"×"关闭按钮，模态框消失
- [ ] 再次打开，点击模态框外部，模态框消失

### 5. 复制功能

**在模态框中**

- [ ] 点击"复制"按钮
- [ ] 按钮变绿显示"已复制！"
- [ ] 2秒后按钮恢复原状
- [ ] 打开记事本或任意文本编辑器，Ctrl+V（或 Cmd+V）
- [ ] 内容正确粘贴

### 6. 填入功能（可选，需要在 AI 平台测试）

**在 ChatGPT 页面（https://chat.openai.com）**

- [ ] 打开插件 → Prompt库
- [ ] 点击任意 Prompt 卡片
- [ ] 点击"填入对话框"按钮
- [ ] 按钮变绿显示"已填入！"
- [ ] ChatGPT 输入框中出现内容
- [ ] 模态框自动关闭

## 🐛 如果还有问题

### 检查浏览器控制台

```
1. 右键点击插件图标
2. 选择"检查弹出内容"
3. 打开 Console 标签页
4. 查看是否有红色错误信息
```

### 常见问题

**问题：Prompt库显示"加载失败"**
- 解决：确保后端运行 `python manage.py runserver`
- 检查 API 地址配置

**问题：没有 Prompt/Template 显示**
- 解决：数据库中没有数据，需要创建测试数据

**问题：填入功能不工作**
- 解决：刷新 AI 平台页面，确保 content script 已加载
