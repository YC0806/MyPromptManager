## 🎬 操作流程演示

### 流程 1：浏览和搜索

```
1. 点击插件图标
   ↓
2. 切换到 "Prompt库" 标签
   ↓
3. 查看卡片列表
   ↓
4. 输入搜索关键词（可选）
   ↓
5. 点击筛选按钮（可选）
   ↓
6. 找到目标 Prompt/Template
```

### 流程 2：复制内容

```
1. 在 Prompt库 中找到目标
   ↓
2. 点击卡片
   ↓
3. 查看详情模态框
   ↓
4. 点击 "复制" 按钮
   ↓
5. 按钮变绿显示 "已复制！"
   ↓
6. 到任何地方粘贴使用
```

### 流程 3：填入对话框

```
1. 访问 AI 平台（ChatGPT/Claude/等）
   ↓
2. 点击插件图标
   ↓
3. 在 Prompt库 中找到目标
   ↓
4. 点击卡片查看详情
   ↓
5. 点击 "填入对话框" 按钮
   ↓
6. 按钮变绿显示 "已填入！"
   ↓
7. 内容出现在 AI 输入框
   ↓
8. 模态框自动关闭
   ↓
9. 编辑并发送消息
```

## 📱 响应式布局

### 固定宽度设计

```
插件弹出窗口：360px 固定宽度
  - 适合大多数屏幕尺寸
  - 不会太宽占用空间
  - 不会太窄影响可读性

内容区域：
  - 列表区域：最大高度 300px，超出滚动
  - 详情内容：最大高度 200px，超出滚动
  - 确保在小屏幕上也能完整显示
```

## 🖱️ 交互提示

### 鼠标操作

```
点击：
  - 标签页按钮 → 切换标签
  - 筛选按钮 → 切换筛选类型
  - 卡片 → 打开详情
  - "复制" 按钮 → 复制内容
  - "填入对话框" → 填入输入框
  - 关闭按钮 (×) → 关闭模态框
  - 模态框外部 → 关闭模态框

悬停：
  - 按钮 → 显示悬停效果
  - 卡片 → 高亮边框

输入：
  - 搜索框 → 实时过滤结果
```

### 键盘操作

```
当前支持：
  - Tab → 在输入框和按钮间切换
  - Enter → 在搜索框中搜索（自动）

未来计划：
  - Esc → 关闭模态框
  - ↑/↓ → 在列表中导航
  - Enter → 打开选中的项目
```
