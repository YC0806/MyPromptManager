# 使用示例

本文档提供 MyPromptManager 浏览器插件的实际使用示例和常见场景。

## 场景 1：快速使用预设 Prompt

### 需求
你在 ChatGPT 中想使用一个预先保存的写作模板来生成文章大纲。

### 步骤

1. **访问 ChatGPT**
   - 打开 https://chat.openai.com 或 https://chatgpt.com
   - 开始一个新对话或打开现有对话

2. **打开插件**
   - 点击浏览器工具栏中的 MyPromptManager 插件图标
   - 切换到"Prompt库"标签页

3. **搜索模板**
   - 在搜索框输入"写作"或"大纲"
   - 系统会实时过滤显示相关的 Prompt 和 Template

4. **选择并填入**
   - 点击找到的"文章大纲生成器"模板卡片
   - 查看完整内容确认是否符合需求
   - 点击"填入对话框"按钮
   - 内容自动填入 ChatGPT 输入框

5. **编辑和发送**
   - 在输入框中根据需要修改内容
   - 按 Enter 或点击发送按钮

### 结果
无需手动复制粘贴，快速使用预设模板开始对话。

## 场景 2：在多个平台使用相同 Prompt

### 需求
你有一个代码审查的 Prompt，想在不同的 AI 平台上测试效果。

### 步骤

1. **在 ChatGPT 测试**
   - 访问 ChatGPT
   - 打开插件 → Prompt库
   - 搜索"代码审查"
   - 点击卡片 → 填入对话框
   - 粘贴你的代码并发送

2. **在 Claude 测试**
   - 访问 https://claude.ai
   - 打开同一个插件（自动识别平台）
   - Prompt库中找到相同的"代码审查"模板
   - 填入对话框（自动适配 Claude 的输入框）
   - 粘贴相同的代码并发送

3. **在 DeepSeek 测试**
   - 访问 https://chat.deepseek.com
   - 重复上述步骤
   - 插件会自动识别并填充 DeepSeek 的输入框

### 结果
无需重复复制，在不同平台快速测试相同的 Prompt，比较效果。

## 场景 3：复制 Template 用于邮件

### 需求
你保存了一个客户沟通的邮件模板，需要复制到邮箱中使用。

### 步骤

1. **打开插件**
   - 在任意页面点击插件图标
   - 切换到"Prompt库"标签页

2. **筛选 Templates**
   - 点击"Templates"筛选按钮
   - 只显示 Template 类型的内容

3. **搜索邮件模板**
   - 输入"客户沟通"或"邮件"
   - 找到对应的模板

4. **复制内容**
   - 点击模板卡片查看详情
   - 点击"复制"按钮
   - 内容已复制到剪贴板

5. **粘贴使用**
   - 打开你的邮箱客户端或 Gmail
   - Ctrl+V (或 Cmd+V) 粘贴内容
   - 根据具体情况修改细节

### 结果
快速访问和复制保存的 Template，无需打开 MyPromptManager 网页版。

## 场景 4：保存和复用对话历史

### 需求
你在 ChatGPT 中与 AI 进行了一次很好的对话，想保存下来并在其他平台继续。

### 步骤

1. **提取对话历史**
   - 在 ChatGPT 对话页面
   - 打开插件 → "对话同步"标签页
   - 点击"提取当前对话"
   - 等待提取完成

2. **在网页版查看**
   - 打开 MyPromptManager 网页版
   - 进入"AI Histories"页面
   - 找到刚才保存的对话
   - 查看完整对话内容

3. **转换为 Template**
   - 在网页版中，将对话的关键部分提取出来
   - 创建一个新的 Template
   - 保存到库中

4. **在其他平台复用**
   - 访问 Claude 或其他 AI 平台
   - 打开插件 → Prompt库
   - 找到刚创建的 Template
   - 填入对话框继续探索

### 结果
知识积累和复用，不同平台间无缝切换。

## 场景 5：团队协作 - 分享 Prompt

### 需求
你的团队在使用统一的 MyPromptManager 服务器，需要共享一些常用的 Prompt。

### 步骤

1. **创建 Prompt（在网页版）**
   - 团队成员 A 在网页版创建 Prompt
   - 添加标签：`团队`、`开发`
   - 保存到共享库

2. **同步到插件**
   - 团队成员 B 在浏览器中打开插件
   - 切换到 Prompt库标签页
   - 系统自动加载最新的 Prompt 列表
   - 可以看到成员 A 创建的 Prompt

3. **使用共享 Prompt**
   - 搜索或筛选 `团队` 标签
   - 找到需要的 Prompt
   - 一键填入使用

4. **持续更新**
   - 任何成员更新 Prompt 后
   - 其他成员刷新插件即可获得最新版本

### 结果
团队协作效率提升，标准化 Prompt 使用。

## 常见技巧

### 快速筛选

1. **按类型筛选**
   - 全部：显示所有内容
   - Prompts：只显示单次使用的提示词
   - Templates：只显示可重复使用的模板

2. **搜索技巧**
   - 搜索标题：输入关键词匹配标题
   - 搜索描述：关键词也会匹配描述内容
   - 实时过滤：输入即搜索，无需按回车

### 键盘快捷操作

- **Tab 切换**：使用鼠标或键盘在"对话同步"和"Prompt库"间切换
- **快速关闭**：在模态框外点击可关闭详情窗口
- **Esc 键**：（未来版本）按 Esc 关闭模态框

### 最佳实践

1. **命名规范**
   - 使用清晰的标题，便于搜索
   - 添加描述说明用途和场景
   - 使用标签分类管理

2. **内容组织**
   - Prompt：用于特定任务的一次性提示
   - Template：可重复使用的结构化模板
   - 定期清理不再使用的内容

3. **跨平台测试**
   - 在不同 AI 平台测试相同 Prompt
   - 记录效果差异
   - 针对平台优化 Prompt

## 故障排除

### 填入失败

**问题**：点击"填入对话框"后没有反应

**解决方法**：
1. 确认当前页面是支持的 AI 提供商
2. 刷新页面后重试
3. 检查浏览器控制台是否有错误信息
4. AI 平台可能更新了页面结构，等待插件更新

### 无法加载 Prompt 列表

**问题**：Prompt库显示"加载失败"

**解决方法**：
1. 检查 MyPromptManager 后端是否运行
2. 验证 API 地址配置是否正确
3. 检查网络连接
4. 在设置中确认 API URL（默认：http://localhost:8000/v1）

### 搜索无结果

**问题**：搜索关键词后无结果显示

**解决方法**：
1. 检查拼写是否正确
2. 尝试使用更通用的关键词
3. 清空搜索框显示全部内容
4. 切换筛选类型（全部/Prompts/Templates）
